## Installing
``` bash
git clone https://github.com/mrprogrammer2938/black-webbrowser

cd black-webbrowser

bash install.sh

python black.py
```

### Argument
```
python black.py 
    --start = Start App
    --url <URL> = Search Website
    --help = Help
    --version = Version
```
